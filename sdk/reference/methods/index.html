    SDK Method Reference¶

    Complete mapping of SDK methods to CLI commands. Click CLI commands to see full documentation.

    Search Methods¶

    SDK Method CLI Command Description
    search_offers() search offers Search available GPU offers
    search_templates() search templates Search templates
    search_volumes() search volumes Search volume offers
    search_network_volumes() search network-volumes Search network volumes
    search_benchmarks() search benchmarks Search benchmarks
    search_invoices() search invoices Search invoices

    Instance Methods¶

    SDK Method CLI Command Description
    show_instances() show instances List all instances
    show_instance(id) show instance Get instance details
    create_instance(id, image, ...) create instance Create instance
    launch_instance(...) launch instance Search and launch in one
    destroy_instance(id) destroy instance Delete instance
    destroy_instances(ids) destroy instances Delete multiple
    start_instance(id) start instance Start stopped instance
    start_instances(ids) start instances Start multiple
    stop_instance(id) stop instance Stop running instance
    stop_instances(ids) stop instances Stop multiple
    reboot_instance(id) reboot instance Reboot instance
    recycle_instance(id) recycle instance Destroy and recreate
    label_instance(id, label) label instance Set instance label
    update_instance(id, ...) update instance Update from template
    prepay_instance(id, amount) prepay instance Add reserved credits
    logs(id, ...) logs Get instance logs
    execute(id, command) execute Run command on instance
